# Insturcciones de la carpeta mapas_nieve/:

Esta carpeta contiene las imágenes para el territorio andorrano clasificadas según la cubierta de nieve: sin datos/nieve/no nieve/nubes remanentes de la serie de años estudiada. 

En el procesado se han borrado las imágenes que no tenían correspondencia con la tabla estadisticas_mapas_nieve.csv y con valores de nubosidad superiores al 10%.
